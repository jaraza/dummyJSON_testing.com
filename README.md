# vovo тестовое задание

Тестирование API авторизации, доступа к данным пользователя и работы с корзиной на dummyjson.com.

### Установка библиотек

pip install -r requirements.txt

### Запуск тестов

cd dummyJSON\_testing.com
pytest

### Баги

Подробно не расписывал, т.к. в MD-файле это выглядит совсем не читабельно

1. Структура products отличается для получения данных корзины и в ответе на добавление новой корзины. В первом случае используется **discountedTotal**, а во втором **discountedPrice**. Выглядит как баг документации. Тест не падает, т.к. написан по документации.
2. При попытке получить данные пользователя используя не валидный токен получаем ошибку **500**, а должна быть **401** . Тест *test\_user\_data\_wrong\_token*
3. При работе с корзинами не проверяет права доступа:

   1. Получение доступа к корзине без авторизации - ***test\_get\_cart\_by\_id\_unauthorized***. Сценарий:

      1. не авторизовываться
      2. отправить запрос на доступ к корзине с id = 1
      3. Ожидаемый результат - код 401
      4. Актуальный результат - код 200, получены данные корзины
   2. Получение доступа к корзинам другого пользователя - ***test\_get\_user\_cart\_wrong\_user***. Сценарий:

      1. авторизоваться используя логин(madisonc) и пароль(madisoncpass) пользователя с id=21, с ролью user
      2. отправить запрос на получение корзин пользователя с id=22 - GET /carts/user/22
      3. Ожидаемый результат - код 403
      4. Актуальный результат - код 200, получены данные корзин пользователя
   3. Создание корзины другому пользователю - ***test\_add\_cart\_wrong\_user***. Сценарий:

      1. авторизоваться используя логин(madisonc) и пароль(madisoncpass) пользователя с id=21, с ролью user
      2. отправить запрос на создаине корзины пользователя с id=22
      3. Ожидаемый результат - код 403
      4. Актуальный результат - код 201, корзина создана для другого пользователя
   4. Обновление корзины другому пользователю - ***test\_update\_cart\_wrong\_user***. Сценарий:

      1. авторизоваться используя логин(madisonc) и пароль(madisoncpass) пользователя с id=21, с ролью user
      2. отправить запрос на обновление корзины пользователя с id=22
      3. Ожидаемый результат - код 403
      4. Актуальный результат - код 200, корзина другого пользователя обновлена
   1. Удаление корзины другому пользователю - ***test\_delete\_cart\_wrong\_user***. Сценарий:

      1. авторизоваться используя логин(madisonc) и пароль(madisoncpass) пользователя с id=21, с ролью user
      2. отправить запрос на удаление корзины пользователя с id=22
      3. Ожидаемый результат - код 403
      4. Актуальный результат - код 200, корзина другого пользователя удалена
